# Security Policy

At SudoWP, we are committed to keeping abandoned WordPress plugins secure.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take security issues seriously. If you have discovered a vulnerability in **Zurich for ClickFunnels**, please do NOT open a public issue.

Instead, please send an email to **security@sudowp.com**.

Please include:
* A description of the vulnerability.
* Steps to reproduce the issue.
* Any relevant proof-of-concept code.

We aim to acknowledge receipt of your vulnerability report within 48 hours and will keep you informed of our progress towards a fix.
